### 柯里化:

?>  `f(x)` 和 `g(x)` 合成为 `f(g(x))`，有一个隐藏的前提，就是f和g都只能接受一个参数。 如果可以接受多个参数，比如 `f(x, y)` 和 `g(a, b, c)`，函数合成就非常麻烦。

```javascript

// 柯里化之前
function add(x, y) {
  return x + y;
}

add(1, 2) // 3

// 柯里化之后
function addX(y) {
  return function (x) {
    return x + y;
  };
}

addX(2)(1) // 3
```
